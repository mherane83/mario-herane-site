# Dr. Mario F. Herane — Bilingual Personal Website

This package contains a richer bilingual personal website in English and Spanish.

## Files

- `index.html` — main website
- `styles.css` — design system and responsive styling
- `script.js` — English/Spanish language toggle
- `google-sites-embed.html` — self-contained HTML version for Google Sites Embed Code
- `assets/mario-herane-hero.jpg` — optimized hero image
- `assets/mario-herane-portrait.jpg` — optimized portrait image
- `assets/resume-mario-herane-march-2026.pdf` — resume PDF
- `sources.md` — public sources and resume notes used to enrich the site

## How to use in Google Sites

### Option 1 — Embed code
1. Open Google Sites.
2. Insert > Embed.
3. Select "Embed code".
4. Open `google-sites-embed.html`.
5. Copy the full HTML and paste it into Google Sites.

Note: The self-contained Google Sites version links the resume button to LinkedIn because Google Sites embeds cannot reliably reference local PDF files. You can change that link after uploading the resume to Google Drive.

### Option 2 — Best visual result
Host this folder on GitHub Pages, Netlify, Vercel, or Google Cloud Storage and embed the hosted URL in Google Sites.

## Recommended edits before publishing

- Add a custom domain if you host externally.
- Replace or add any missing publications you want featured.
- Upload the resume to Google Drive and replace the resume link in `google-sites-embed.html`.
- Consider adding a "Media / Press" section later if you want a stronger executive brand presence.
